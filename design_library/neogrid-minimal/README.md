# NeoGrid Minimal - Design System

## Overview
A dark, highly functional, geometric design system emphasizing clean lines, grid structure, and vibrant neon accents for a modern, futuristic synthesizer interface.

## Palette
- **Primary**: #00FFFF
- **Secondary**: #FF0099
- **Background**: #111115
- **Surface**: #1E1E24

## Components
- Knob Style: futuristic
- Fader Style: flat
- Border Radius: 4px
